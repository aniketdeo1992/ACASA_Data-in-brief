# ACASA_Data-in-brief
This repository is an additional supplementary material for the article :  A Harmonized Gridded Input Dataset for Simulation Modelling of the Major Crops of South Asia published in "Data in Brief" journal.
Folder "" contains CSV files to compare outputs of water holding capacity, nitrogen consumption, and understand irrigation share and typology in Rice.
Folder "" contains an intermediary file (CSV) used for nitrogen consumption calculation. This folder also contains the MATALB script to generate gridded nitrogen consumption data from this intermediary file.
